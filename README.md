# fire dree (HTML5 Game)

A basic 2D shooting game inspired by Free Fire, made with HTML5 and JavaScript.  
Click to shoot, use arrow keys to move. Simple AI enemies follow you.

## Controls

- Move: Arrow keys (↑ ↓ ← →)
- Shoot: Mouse click

## Features

✅ Player movement  
✅ Shooting  
✅ Enemies  
✅ Collision detection  
✅ Responsive canvas
